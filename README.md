# Interactive Data Table

A client-side JavaScript library for interactive, explorable data tables using DuckDB WASM for in-browser analytics.

## Features

- **Complete Privacy**: All data processing happens in the browser - no server-side processing
- **DuckDB WASM**: Powerful SQL analytics engine running in a Web Worker
- **Interactive Visualizations**: Histograms, value counts, and more in column headers
- **Crossfilter Support**: Filter on any column and see updates across all visualizations
- **Virtual Scrolling**: Smooth handling of large datasets
- **Export Options**: CSV, JSON, and clipboard support

## Installation

```bash
npm install @jeyabbalas/data-table
```

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm test
```
